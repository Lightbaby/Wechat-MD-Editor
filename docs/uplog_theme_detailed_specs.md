# UPlog 多风格主题视觉设计规范 (High-Definition Specs)

本文档旨在对 UPlog 的 16 款主题进行**像素级**的视觉拆解。所有描述基于对模版三张预览图（封面、正文、结尾）的微距观察。

---

## 01. 简单格子 (Simple Grid)

**1. 核心视觉 (Core Visuals)**
*   **整体风格**: 极简主义、理性、干净、手账风。
*   **背景细节**:
    *   底色为纯白 (#FFFFFF)。
    *   覆盖全屏的**浅灰色虚线格纹** (Dashed Grid)，线条颜色极浅 (#F0F0F0 或 #E5E5E5)，网格间距较大（约 40px），营造出一种“绘图纸”或“方眼纸”的质感。
*   **配色方案**:
    *   **主色**: 纯黑 (#000000) 用于标题；深灰 (#333333) 用于正文。
    *   **强调色**: 无明显彩色，主要靠黑白灰对比。
    *   **装饰色**: 浅灰 (#D9D9D9) 用于背景元素。

**2. 排版规范 (Typography)**
*   **字体**: 无衬线字体 (Sans-serif)，类似 Inter 或 PingFang SC，字形方正，现代感强。
*   **标题 H1**:
    *   居中对齐 (Centered)。
    *   字号极大 (约 48px+)，字重 Bold。
    *   颜色：纯黑。
*   **标题 H2**:
    *   左对齐。
    *   **装饰前缀**: 标题文字前有一个**黑色实心圆角正方形** (Rounded Square)，尺寸约与文字高度一致，颜色 #000000，与文字间距约 8px。
*   **标题 H3**:
    *   **装饰前缀**: 黑色实心菱形 (◆) 或圆点，尺寸较小。
*   **正文 Body**:
    *   行高宽松 (1.6-1.8)，字重 Regular，颜色 #333333。

**3. 组件样式 (Components)**
*   **页眉 (Header)**:
    *   右上角悬浮着类似 **"四角星/闪烁" (Sparkles)** 的装饰图标，颜色为极浅的灰色，作为背景点缀。
*   **列表 List**:
    *   实心圆点 (•)，颜色黑色。
*   **强调 Emphasis**:
    *   单纯加粗，无颜色变化。
*   **布局 Layout**:
    *   内容区域留白较大，四周有明显的 Padding，通透感强。

## 02. 彩页随笔 (Color Page Notes)

**1. 核心视觉 (Core Visuals)**
*   **整体风格**: 梦幻、温柔、手绘感、少女心。
*   **背景细节**:
    *   **全屏柔光渐变** (Soft Gradient)。
    *   Variant 1: 浅蓝 -> 浅绿 (Blue-Green)。
    *   Variant 2: 浅粉 -> 浅黄 (Pink-Yellow)。
    *   Variant 3: 浅紫 -> 浅蓝 (Purple-Blue)。
    *   在渐变之上，散布着**手绘风格的空心/实心菱形**和**四角星**装饰，颜色为半透明的白色。
*   **配色方案**:
    *   **主色**: 墨绿/深灰 (#2F4F4F) 用于文字，避免纯黑的死板。
    *   **背景**: 高明度、低饱和度的马卡龙色系渐变。

**2. 排版规范 (Typography)**
*   **字体**: **衬线字体** (Serif / Songti)，类似“思源宋体”或“全宋体”，笔画有粗细变化，强化文学和手记的氛围。
*   **标题 H1**:
    *   左对齐/居中。
    *   字重 Bold，但因是宋体显得优雅。
*   **标题 H2**:
    *   **装饰前缀**: **实心菱形** (◆) 图标，颜色与当前主题的主色调一致（如深绿色），尺寸适中。
*   **标题 H3**:
    *   **装饰前缀**: 较小的菱形或加粗处理。

**3. 组件样式 (Components)**
*   **列表 List**:
    *   **彩色实心菱形** (◆) 作为列表符号，而非圆点，为了呼应背景装饰。
*   **卡片/容器**:
    *   无明显的边框容器，文字直接浮在渐变背景上，利用阴影或文字颜色保证可读性。

## 03. 纸纹背景 (Paper Texture)

**1. 核心视觉 (Core Visuals)**
*   **整体风格**: 质感、厚重、怀旧、模拟真实材质。
*   **背景细节**:
    *   **深色高清纹理**。
    *   Texture 1: **黑板/深灰岩石** (Blackboard/Slate)，有粉笔灰或石头颗粒感。
    *   Texture 2: **深棕色粗糙纸张** (Rough Craft Paper)。
    *   Texture 3: **深色布纹/亚麻** (Dark Linen)。
*   **配色方案**:
    *   **主色**: **纯白** (#FFFFFF) 或 极浅米色 (#F5F5DC)。在深色纹理上形成极强对比。
    *   **背景**: 深色系纹理图片。

**2. 排版规范 (Typography)**
*   **字体**: **衬线字体** (Serif)，白色宋体在深色纹理上显得非常有文化沉淀感（类似电影字幕）。
*   **标题 H1**:
    *   居中对齐。
    *   字号大，白色，可能有轻微的**文字阴影** (Drop Shadow) 以提升在纹理上的可读性。
*   **标题 H2**:
    *   **装饰前缀**: 复杂的**古典花纹符号**或**星号** (*)，颜色为白色或浅金。
*   **标题 H3**:
    *   通常为加粗的白色文字，可能带有下划线。

**3. 组件样式 (Components)**
*   **装饰元素**:
    *   页面四周可能有**暗角** (Vignette) 效果，聚焦中心内容。
*   **列表 List**:
    *   白色实心圆点或短横线。

## 04. 备忘录 (Memo)

**1. 核心视觉 (Core Visuals)**
*   **整体风格**: 拟物化、办公、效率、便签条。
*   **背景细节**:
    *   **横线信纸** (Ruled Paper)。背景是白色或浅米黄 (#FFFDF0)，上面铺满了**浅灰色虚线/实线**，行距固定（约 50-60px），模拟真实的横格笔记本。
*   **配色方案**:
    *   **主色**: 深灰/铅笔灰 (#424242)。
    *   **强调色**: **荧光黄** (#FFF176) 或 **便签红**，用于马克笔涂抹效果。

**2. 排版规范 (Typography)**
*   **字体**: **等宽字体** (Monospace) 或 手写体/圆体，模仿打字机或手写笔记的痕迹。
*   **标题 H1**:
    *   通常位于页面最上方，不一定很大，但位置显著。
*   **标题 H2**:
    *   **高亮装饰**: 核心特征是**马克笔高亮** (Highlighter Effect)。文字背景有一块不规则的、色彩鲜艳的（黄/粉/绿）色块，模拟荧光笔划过的痕迹。
*   **标题 H3**:
    *   加粗，可能带有下划线。

**3. 组件样式 (Components)**
*   **页眉 (Header)**:
    *   **拟物化表单头**: 顶部有明显的 `DATE: _______` 和 `MEMO NO: _______` 字样，字体为极细的衬线体或手写体，颜色浅灰，极具辨识度。
*   **列表 List**:
    *   **红色/彩色实心圆点**。在黑白为主的信纸上，红点非常显眼，像老师的批注。
*   **布局**:
    *   文字严格对齐背景的横线（理想状态下）。

## 05. 神奇物语 (Magic Story)

**1. 核心视觉 (Core Visuals)**
*   **整体风格**: 塔罗牌、神秘学、古典魔法书、哈利波特风格。
*   **背景细节**:
    *   **紫色复古纸张** (Purple Vintage Paper)。背景是有颗粒感的浅紫色 (#E6E6FA 变调) 或 羊皮纸纹理。
*   **装饰框 (Frame)**:
    *   页面四周有一个显著的**黑色古典花纹边框**，四角有复杂的 **Corner Ornaments** (卷草纹/欧式花纹)。这是该主题的视觉核心。
*   **配色方案**:
    *   **主色**: 黑色 (#1A1A1A) 用于文字和边框。
    *   **背景**:  dusty purple / misty rose。

**2. 排版规范 (Typography)**
*   **字体**: **衬线字体** (Serif)，具有古典韵味。
*   **标题 H1**:
    *   居中对齐。
    *   字号大，字重 Regular。
*   **标题 H2**:
    *   **装饰容器**: 标题数字（如"01"）被包裹在一个**圆形的魔法徽章** (Magic Badge) 中，徽章边缘有齿轮或光芒状装饰。
    *   文字居中或紧随徽章。
*   **标题 H3**:
    *   **装饰前缀**: 实心菱形或魔法符号 (✧)。

**3. 组件样式 (Components)**
*   **列表 List**:
    *   **菱形/星形符号** (✦)，黑色，与魔法主题呼应。
*   **分割线**:
    *   可能会使用花纹分割线 (Ornamental Divider)。

## 06. 中国色 (China Color)

**1. 核心视觉 (Core Visuals)**
*   **整体风格**: 东方美学、传统色彩、沉稳、文化感。
*   **背景细节**:
    *   **纯色底 + 隐纹**。
    *   Variant 1: **靛蓝/青花蓝** (Indigo)。
    *   Variant 2: **朱红/故宫红** (Vermilion)。
    *   Variant 3: **竹青/墨绿** (Dark Green)。
    *   背景可能有极淡的宣纸纹理或云纹水印。
*   **配色方案**:
    *   **主色**: **纯白** (#FFFFFF) 用于所有文字，以在深色背景上凸显。
    *   **背景**: 高饱和度、低明度的中国传统色。

**2. 排版规范 (Typography)**
*   **字体**: **衬线字体** (Serif / Songti)，宋体是必须的，传递书卷气。
*   **标题 H1**:
    *   **竖排/复古布局** (可能)：在某些变体中，标题可能尝试竖排或使用特殊的徽章式布局，但在标准 Markdown 渲染中通常为横排大字，居中或左对齐。
*   **标题 H2**:
    *   **装饰前缀**: **空心/实心圆形**，或者带有传统回纹装饰的序号。
    *   可能带有上下边框线条。

**3. 组件样式 (Components)**
*   **装饰元素**:
    *   角落可能有 **祥云** (Cloud Pattern) 或 **印章** (Seal) 风格的 SVG 装饰。
*   **列表 List**:
    *   白色实心圆点。
*   **引用 Quote**:
    *   可能会有“竖线”装饰，模仿古籍的版式。

## 07. 撕拉便签 (Torn Note)

**1. 核心视觉 (Core Visuals)**
*   **整体风格**: 拼贴艺术、手账、随性、生活化。
*   **背景细节 (Body Background)**:
    *   整个页面背景是深色木纹或灰色的桌面纹理。
*   **内容容器 (Container)**:
    *   核心内容区域是一张**白色便签纸**，四周边缘是**不规则的锯齿状撕裂效果** (Torn Edges/Jagged Edges)。
    *   便签纸有轻微的**投影** (Drop Shadow)，营造悬浮在桌面上的立体感。
*   **配色方案**:
    *   **主色**: 深咖啡色 (#4E342E) 或 黑色。
    *   **各种胶带色**: 装饰元素模拟胶带。

**2. 排版规范 (Typography)**
*   **字体**: **衬线字体** 或 手写体，配合纸张质感。
*   **标题 H2**:
    *   **笔刷背景**: 标题文字背后有一道**油漆刷/马克笔刷痕** (Brush Stroke)，颜色通常为棕色或浅咖色，文字为白色 (反白)。
*   **标题 H3**:
    *   可能带有黄色荧光笔高亮。

**3. 组件样式 (Components)**
*   **列表 List**:
    *   深色实心圆点。
*   **装饰**:
    *   顶部可能模拟“胶带”粘贴的效果。

## 08. 一页信笺 (Letterhead)

**1. 核心视觉 (Core Visuals)**
*   **整体风格**: 商务、精致、杂志、优雅。
*   **背景细节**:
    *   **米色/信纸黄** (#FAF9F6) 渐变或纹理。
    *   Color 1 looks like a soft "Morning Mist" gradient (Blue to Pink/Yellow).
    *   核心特征是顶部有一张**全宽 Hero Image** (头图)，且头图通常是不规则形状或带有蒙版。
*   **配色方案**:
    *   **主色**: 甚至黑 (#2C2C2C)。
    *   **强调色**: 莫兰迪色系。

**2. 排版规范 (Typography)**
*   **字体**: **衬线字体** (Serif)，精致且字间距稍大。
*   **标题 H1**:
    *   通常位于 Hero Image 下方，居中或左对齐。
*   **标题 H2**:
    *   **序号装饰**: 黑色实心圆圈 + 白色数字 (➊, ➋)，或者圆圈在左侧。
*   **标题 H3**:
    *   带有特定颜色的竖线前缀。

**3. 组件样式 (Components)**
*   **页眉 (Header)**:
    *   **模拟 App 导航栏**: 顶部有一个明显的导航条，左侧有 "← Back"，中间是 "UPlog"，右侧是 "..."，模拟手机 App 的界面，给人一种“原生”的感觉。
*   **列表 List**:
    *   精致的小圆点。

## 09. 流光像素 (Pixel Stream)

**1. 核心视觉 (Core Visuals)**
*   **整体风格**: 学院风、英伦菱格、几何、理性。
*   **背景细节**:
    *   **菱形格纹** (Argyle Pattern / Diamond Grid)。背景铺满淡雅的菱形交叉纹理。
    *   Color 1: 浅蓝底 + 白色线条。
    *   Color 2: 浅灰紫底 + 白色线条。
*   **配色方案**:
    *   **主色**: 黑色/深空灰。
    *   **背景**: 极浅的蓝/紫/粉色系。

**2. 排版规范 (Typography)**
*   **字体**: **无衬线字体** (Sans-serif)，干净利落。
*   **标题 H1**:
    *   居中，黑色。
*   **标题 H2**:
    *   **装饰前缀**: **黑色实心正方形**，内含反白数字 (如 "01")。
    *   数字与标题文字紧密排列。
*   **标题 H3**:
    *   **装饰前缀**: 粗壮的**竖线** (|)，颜色通常为黑色或深灰。

**3. 组件样式 (Components)**
*   **列表 List**:
    *   实心圆点，黑色。
*   **装饰**:
    *   背景的菱形纹理本身就是最大的装饰，无需额外元素。

## 10. 酷炫卡片 (Cool Card)

**1. 核心视觉 (Core Visuals)**
*   **整体风格**: 潮流、年轻、高饱和度、卡片式 UI。
*   **背景细节**:
    *   **全屏高饱和渐变** (Vibrant Gradient)。
    *   Color 1: 蓝紫 -> 玫红 (Blue-Purple-Pink)。
    *   Color 2: 青蓝 -> 嫩绿。
*   **内容容器 (Container)**:
    *   内容承载于一张居中的**白色圆角卡片** (White Rounded Card) 上。
    *   卡片有明显的**弥散投影** (Drop Shadow)，制造悬浮感。
*   **配色方案**:
    *   **主色**: 黑色（在白卡片上）。
    *   **强调色**: 与背景渐变一致的亮色（如玫红/亮蓝）。

**2. 排版规范 (Typography)**
*   **字体**: **无衬线字体** (Sans-serif)，现代且粗壮。
*   **标题 H2**:
    *   **底部高亮**: 标题文字下方有一条**粗荧光下划线/笔刷** (Thick Underline)，颜色为亮粉色或亮青色 (Hex #FF00FF approx)，只覆盖文字宽度的 100%。
    *   或者直接使用文字颜色渐变。
*   **标题 H3**:
    *   **装饰前缀**: 竖线，颜色可能跟随强调色。

**3. 组件样式 (Components)**
*   **列表 List**:
    *   普通黑点。

## 11. 现代纹理 (Modern Texture)

**1. 核心视觉 (Core Visuals)**
*   **整体风格**: 商务、暗黑模式、高端、科技感。
*   **背景细节**:
    *   **深色抽象纹理**。
    *   可能是深灰色的拉丝金属、碳纤维或抽象几何暗纹。
*   **配色方案**:
    *   **主色**: **纯白** (#FFFFFF) 用于正文。
    *   **背景**: 深黑/深灰 (#111111)。
    *   **强调色**: **亮黄色** (#FFD700) 或 **霓虹绿**。

**2. 排版规范 (Typography)**
*   **字体**: **无衬线字体** (Sans-serif)。
*   **标题 H2**:
    *   **黄色高亮**: 标题文字本身颜色为**亮黄色**。
    *   **装饰前缀**: **黄色竖线** (|) 或 黄色方块。
*   **标题 H3**:
    *   白色加粗，或带有黄色箭头。

**3. 组件样式 (Components)**
*   **列表 List**:
    *   白色或黄色圆点。

## 12. 复古通用 (Retro Universal)

**1. 核心视觉 (Core Visuals)**
*   **整体风格**: 经典、书香、人文、阅读。
*   **背景细节**:
    *   **米黄色帆布/亚麻纹理** (Canvas/Linen Texture)。质感细腻，色调温暖 (#FDF5E6)。
*   **Hero Image**:
    *   顶部通常配有一张充满人文气息的图片（如书籍、咖啡）。
*   **配色方案**:
    *   **主色**: 深褐/黑色。
    *   **背景**: 米黄纹理。

**2. 排版规范 (Typography)**
*   **字体**: **衬线字体** (Serif / Songti)。
*   **标题 H2**:
    *   **下划线装饰**: 标题下方有一条黑色的**粗实线** (Thick Line)，长度与文字相当或稍长。
    *   序号 (如 "01") 字号较大，位于标题上方或左侧，形成层级。
*   **标题 H3**:
    *   无过多装饰，靠字体加粗区分。

**3. 组件样式 (Components)**
*   **列表 List**:
    *   **空心菱形** (◇) 作为列表符号，显得非常精致。
*   **引用 Quote**:
    *   左侧有粗黑线边框，经典的引言样式。

## 13. 人文叙事 (Humanistic Narrative)

**1. 核心视觉 (Core Visuals)**
*   **整体风格**: 博客、像 Medium 或 少数派、阅读感强、知性。
*   **背景细节**:
    *   **暖白/米白背景** (#F9F9F5)。非纯白，保护视力，适合长文阅读。
*   **配色方案**:
    *   **主色**: **古铜色/深棕色** (Bronze/Dark Brown, #8B4513)。用于标题和强调。
    *   **正文**: 深灰。

**2. 排版规范 (Typography)**
*   **字体**: **衬线字体** (Serif) 用于标题，无衬线用于正文，或者全衬线。
*   **标题 H2**:
    *   **竖线前缀**: 标题左侧有**深棕色粗竖线** (Thick Vertical Bar)，高度与文字对齐。
    *   文字颜色为深棕色。
*   **标题 H3**:
    *   可能仅加粗，或带有简单的圆形符号。

**3. 组件样式 (Components)**
*   **引用 Quote**:
