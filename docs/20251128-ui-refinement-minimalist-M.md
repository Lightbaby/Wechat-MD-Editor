# UI 极简优化 - Less is More

## 📋 任务信息
- 类型：🟡 中等任务 (M)
- 预计时间：1 小时
- 创建时间：2025-11-28
- 来源：用户反馈（"太丑"、"一眼AI"、"Less is More"）

## 🎯 目标
去除繁杂的装饰（阴影、圆角、渐变、高亮色块），回归内容的本质。打造 **"Clean, Crisp, Professional"** 的界面。

## 🎨 设计原则
1.  **去色 (De-color)**：UI 框架去色，只保留黑白灰，让用户的文章和主题成为视觉焦点。
2.  **去噪 (De-noise)**：移除多余的阴影 (`shadow-xl`)、厚重的边框 (`border-2`) 和复杂的背景色。
3.  **精致 (Refined)**：使用更细的线条 (`border`, `stroke-width: 1.5`)，更克制的圆角 (`rounded-lg`)。

## 🔧 修改方案

### 1. 侧边栏 (Sidebar)
-   **旧**：深色背景 (`bg-slate-900`)，高亮色块 (`bg-purple-600`)。
-   **新**：
    -   白底 (`bg-white`) + 极细右边框 (`border-gray-100`)。
    -   图标：默认灰 (`text-gray-400`)，激活/悬停黑 (`text-gray-900`)。
    -   移除所有彩色背景块，仅通过图标颜色或极简的指示器表示状态。

### 2. 侧滑面板 (Drawer)
-   **旧**：多彩的 AI 按钮 (`bg-emerald-50` 等)。
-   **新**：
    -   纯白背景。
    -   AI 功能列表改为**极简列表项**，仅在 Hover 时出现浅灰背景。
    -   移除大色块图标背景。

### 3. 主题面板 (Theme Panel)
-   **旧**：粗边框 (`border-2`)，Hover 缩放。
-   **新**：
    -   细边框 (`border`)，移除缩放动画。
    -   简化卡片内容，专注于配色预览。

### 4. 编辑器 & 预览
-   移除手机外壳的厚重阴影，改为轻微的 `shadow-sm` 或纯边框。
-   顶部面包屑导航更加极简。

## ✅ 验收标准
- [ ] 界面无大面积高饱和度颜色。
- [ ] 侧边栏融入背景，不突兀。
- [ ] 整体质感接近 Notion / Linear 等现代生产力工具。
